# Philippians Commentary

A static, responsive study site for the four chapters of Philippians, with
the King James text, verse-by-verse commentary, original-language notes,
cross-reference previews, and a historical introduction.

## Development

```bash
npm ci
npm run dev
```

## Quality checks

```bash
npm run validate
npm run typecheck
npm run lint
npm run build
```

Deployment instructions are in [DEPLOYMENT.md](./DEPLOYMENT.md).
